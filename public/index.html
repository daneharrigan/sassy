Sass as a Service

A proof of concept to remove the Ruby dependency from designers wanting to use Sass and Compass.

Using Sass

Set the style tag's type attribute to text/sass. 
<!doctype html>
<script src="http://sassy.herokuapp.com/sassy.js"></script>
<style type="text/sass">
$light-grey: #eee
$dark-grey: #333

body
  background: $light-grey
  color: $dark-grey

  p
    font-size: 14px
</style>

<!-- your html -->

Using SCSS

Set the style tag's type attribute to text/scss.

<script src="http://sassy.herokuapp.com/sassy.js"></script>
